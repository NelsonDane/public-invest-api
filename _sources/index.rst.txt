Unofficial Public Invest API Docs
==================================
Welcome! This is an unofficial API for Public.com, a popular investing site. It is created using a Python wrapper around the Public.com requests API that the website uses.
This project is not affiliated with Public.com in any way, so use at your own risk.

This project is still a work in progress, so it might have bugs or missing features. Feel free to contribute or Sponsor!
