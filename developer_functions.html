Developer Functions¶

These are functions that are used internally by the library and are not meant to be used by the end user. If you add support for a new endpoint, then you’ll have to add a new function here! (in the endpoints.py file)

Bases: object

Initializes the Public.com API client.

Parameters:

  • filename – The filename to save the cookies to. Defaults to “public_credentials.pkl”.

  • path – The path to save the cookies to. Defaults to current directory.
